## ЧАСТЫЕ ВОПРОСЫ НА ДЕМО

**"А если AI ответит неправильно?"**
> "Отличный вопрос. Система устроена так: сначала FAQ с высокой уверенностью, потом база знаний, потом GPT. Если уверенность ниже 25% — система не гадает, а говорит клиенту: 'позвоните на 1200'. Нет галлюцинаций по финансовым данным."

**"Насколько это безопасно?"**
> "Данные клиентов не уходят в OpenAI — только текст вопроса, без персональных данных. IP хранится в виде хеша. Domain Guard блокирует нерелевантные запросы. Rate Limiter защищает от спама."

**"Можно заменить OpenAI?"**
> "Да, один файл openai.ts. Замените URL — и система работает с любым OpenAI-совместимым API: Ollama локально, Azure OpenAI, Yandex GPT."
