## КАК ЗАПУСТИТЬ ПРИЛОЖЕНИЕ ЛОКАЛЬНО

```bash
# 1. Запустить backend (в первом терминале)
cd /Users/azizbekkhabibullaev/Downloads/bank-chatbot-master/apps/backend
node --require dotenv/config src/server.ts
# Или: npx ts-node src/server.ts
# Должен запуститься на http://localhost:3000

# 2. Запустить frontend (во втором терминале)
cd /Users/azizbekkhabibullaev/Downloads/bank-chatbot-master/apps/widget
npm run dev
# Должен запуститься на http://localhost:5173

# 3. Открыть браузер:
# Виджет: http://localhost:5173
# Виджет (demo mode): http://localhost:5173/?demo=1
# Admin: http://localhost:5173/admin
```

**Учётные данные admin:**
- Логин: `admin`
- Пароль: задаётся через `ADMIN_PASSWORD` в .env

## СОВЕТЫ ПО КАЧЕСТВУ СКРИНШОТОВ

1. **Браузер:** Chrome / Firefox в режиме инкогнито (без расширений)
2. **Масштаб:** 100% (не zoom in/out)
3. **Разрешение:** Минимум 1440×900
4. **Формат:** PNG (не JPG — лучше качество текста)
5. **Для admin:** Убедиться что есть тестовые данные (лиды, диалоги)
6. **Demo mode:** `http://localhost:5173/?demo=1` автоматически запустит 3 демо-сообщения

**Для добавления тестовых данных:**
```bash
cd /Users/azizbekkhabibullaev/Downloads/bank-chatbot-master
npx ts-node scripts/seed-demo.ts
```
